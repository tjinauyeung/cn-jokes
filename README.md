## Project
A simple Chuck Norris list of jokes app

#### Dependencies
- Node.js
- NVM

#### Getting started

1. Use configured Node version

```bash
nvm use
```

2. Install npm packages

```bash
npm i
```

3. Start project

```bash
npm start
```

Visit localhost:1234 to see project.

#### Run tests

Run test once
```bash
npm test
```

Watch tests
```bash
npm test:watch
```
